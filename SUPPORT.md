# Support

Support for this project is handled through GitHub issues.

## Best Places To Ask

- Bug report: open a bug issue with your environment and traceback.
- Compatibility question: open a compatibility issue with Python, Neo4j, and py2neo versions.
- Sample data question: use `Invoice_data_Demo.xls` first, then open an issue if the sample does not work.
- Modernization request: use the dependency and compatibility modernization tracking issue.

## Not Supported

- Private debugging by email.
- Production incident response.
- Unrelated Neo4j, pandas, or Python installation support outside this example project.
- Requests that do not include enough environment information to reproduce the problem.
